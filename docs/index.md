# Analisis Pengucapan Kata "Very"

Website dokumentasi untuk project analisis pengucapan kata *very* 
menggunakan pendekatan deep learning.

## Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis pengucapan kata *very*
menggunakan audio speech dan visualisasi Mel-Spectrogram.

## Teknologi
- Python
- Streamlit
- Librosa
- GitHub Actions
- MkDocs
